## Submission Tips

1. **One per day** - Don't spam. 2-3 directories per day is fine.
2. **Unique descriptions** - Use the variations in `directory-listings.md`
3. **Photos** - Upload 5+ professional photos to each
4. **Verify immediately** - Most send email/phone verification
5. **Track everything** - Update this file as you go
